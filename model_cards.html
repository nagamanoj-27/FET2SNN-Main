🌐 How the Website Works
Architecture Overview

FET2SNN is a fully client-side, single-page application (SPA) built in pure HTML5, CSS3, and vanilla JavaScript — no backend server, no Python runtime, no API calls during use. All computation runs live in the browser using hand-coded analytical physics models.

Data Flow — Every Slider Move
1
User moves a slider
An input event fires on the range element. The global state object S is updated with the new value (e.g., S.Lg = 12). The paired number input is also synced.
2
computeAll() is called
This master function orchestrates the full pipeline: device physics → SNN prediction → all chart updates → 3D model rebuild → band diagram → raster plot. Everything runs synchronously in < 5 ms on modern hardware.
3
computeDevice(S) → FET parameters
Runs the full analytical physics engine (see §2). Returns an object with Weff, Cox, Vth, SS, DIBL, µeff, Ion, Ioff, gm, gds, Vov, β.
4
computeSNN(d, S) → neuromorphic metrics
Feeds the device parameters into the LightGBM-surrogate linear model. Returns Energy/spike (fJ), Inference Latency (ms), SNN Accuracy (%).
5
All 11 charts update in-place
Chart.js datasets are replaced and .update() is called with animation.duration = 0 (instant re-render) for real-time responsiveness.
6
Three.js model rebuilds
window.update3DModel() tears down the previous deviceGroup, creates fresh geometries scaled to the current Lg, Wns, Tns, Nstacks values, and adds them back to the scene. The animation loop rerenders at ~60 fps.
7
Band diagram & raster update
updateBandDiagram(Vth, Vdd) redraws the Canvas 2D band structure. updateRasterParams(E, L) adjusts spike probability and dot size in the LIF neuron raster animation loop.

⚛️ Device Physics Engine — computeDevice(S)
Weff Effective Gate Width
\[ W_{\text{eff}} = N_{\text{stacks}} \times 2 \times (W_{ns} + T_{ns}) \quad \text{[nm]} \]
Ref: Gate-all-around wrap-around perimeter model (TSMC / Samsung GAA)

Each nanosheet contributes two flat faces (top + bottom) plus two thin side edges. The factor of 2 accounts for the full wraparound gate perimeter. Units: nm → converted to m internally via Wm = Weff × 10⁻⁹.

Cox Oxide Capacitance
\[ C_{ox} = \frac{\varepsilon_0 \cdot 3.9}{\text{EOT}} = \frac{34.5}{\text{EOT}} \quad \text{[fF/μm²]} \]
ε₀ = 8.854×10⁻¹² F/m · κ_SiO₂ = 3.9 → 34.5 fF/μm² at EOT = 1 nm

Internally converted to SI: Cox_Fm2 = Cox × 10⁻³ F/m².

Vth Threshold Voltage

Built up from four additive components:

\[ V_{th,25} = \underbrace{\left(0.3 + 0.02\frac{W_{ns}-20}{10} - 0.01(T_{ns}-5) + \Delta V_{mat}\right)}_{V_{th0}} + \underbrace{0.04\left(\frac{5}{T_{ns}}\right)^2}_{\Delta V_{QM}} + \underbrace{0.24\left(\frac{L_g}{12}-1\right) \cdot f_{arch}}_{\Delta V_{roll}} + \underbrace{0.002(k_{sp}-7) \cdot f_{arch}}_{\Delta V_{sp}} \]
Sentaurus-calibrated SCE roll-off; Martino 2025 temperature correction; Aruna Kumari 2025 material shifts
TermFormulaMeaning
Vth₀0.3 + 0.02(Wns−20)/10 − 0.01(Tns−5) + ΔVmatNominal Vth at Lg=12, tuned to geometry
ΔV_QM0.04 · (5/Tns)²Quantum confinement blue-shift (thinner sheet → higher Vth)
ΔV_roll0.24·(Lg/12−1)·archFSCE roll-off (short channel → lower Vth)
ΔV_sp0.002·(κ−7)·archFSpacer fringe field effect on Vth
\[ V_{th}(T) = V_{th,25} \cdot C_{\text{corner}} - \alpha(T-25), \quad \alpha = 0.0005\,\text{V/°C} \]

Material offsets ΔVmat: Si = 0, VP = +0.15 V, BP = +0.05 V, SiGe = −0.10 V, InGaAs = −0.15 V.

Clamped to [0.2, 0.5] V.

SS Subthreshold Swing
\[ SS_0 = 60 + 1.2\left(\frac{L_g - 12}{12}\right)^2 \quad \text{[mV/dec]} \]
\[ SS_{25} = SS_0 \cdot \left(1 + 0.02\,(k_{sp}-7)\cdot f_{arch}\right) \cdot f_{arch} \]
\[ SS(T) = SS_{25} + \beta\,(T - 25), \quad \beta = 0.19\,\text{(mV/dec)/°C} \]
Boltzmann limit = 60 mV/dec; SCE penalty via (Lg−12)² term; Lombardi spacer correction

SS rises with temperature because the thermal voltage kT/q increases. The spacer architecture factor archF (triple=0.75, dual=0.85, single=1.0) reduces SCE-induced SS degradation. Clamped to [60, 100] mV/dec.

DIBL Drain-Induced Barrier Lowering
\[ \text{DIBL}_0 = 30 \cdot \left(\frac{12}{L_g}\right)^{1.2} \quad \text{[mV/V]} \]
\[ \text{DIBL} = \text{DIBL}_0 \cdot \left(1 + 0.05\,(k_{sp}-7)\cdot f_{arch}\right) \cdot \left(1 + 0.002\,(T-25)\right) \cdot f_{arch} \]
Power-law SCE scaling; Sentaurus fit to 17-device benchmark

DIBL increases sharply for shorter channels (power exponent 1.2). Higher κ spacer increases DIBL due to enhanced fringing. Temperature increases DIBL slightly. Clamped to [10, 80] mV/V.

µeff Effective Mobility
\[ \mu_{\text{eff}} = \frac{\mu_0}{1 + 0.03\,(V_{dd}-0.7)} \cdot \left(\frac{300}{T+273}\right)^{1.5} \cdot \frac{1}{1 + 1.2\,(5/T_{ns})^2} \]
Lombardi transverse-field degradation · phonon scattering (T³/²) · MLDA quantum confinement
MaterialBase µ₀ (cm²/V·s)ΔVmat (V)
Silicon (Si)2000
Violet Phosphorene (VP)150+0.15
Black Phosphorene (BP)300+0.05
SiGe (Strained)350−0.10
InGaAs (High Mob)800−0.15

The confinement correction 1/(1 + 1.2·(5/Tns)²) models the MLDA (Modified Local Density Approximation) quantum potential effect from Sentaurus Device — reduces µeff by ~55% for Tns = 5 nm vs bulk.

Ion On-Current (Velocity Saturation Model)
\[ \beta = \mu_{\text{eff}} \cdot C_{ox} \cdot \frac{W_{\text{eff}}}{L_g} \]
\[ V_{dsat} = \frac{V_{ov}}{1 + V_{ov}/(E_{sat}\cdot L_g)}, \quad E_{sat} = 10^7\,\text{V/m} \]
\[ I_{on} = \frac{\beta\,V_{ov}^2}{1 + V_{ov}/(E_{sat}\cdot L_g)} \quad \text{[A]} \]
Two-piece velocity saturation model (Tsividis); Esat = 10⁷ V/m for Si NMOS

Where Vov = Vdd − Vth (overdrive). The velocity saturation denominator (1 + Vov/Esat·Lg) prevents unphysical super-linear Ion growth at short channels. In the linear region (Vds < Vdsat):

\[ I_D\,(V_{gs}, V_{ds}) = \frac{\beta\,(2V_{ov}-V_{ds})\,V_{ds}}{2\,(1 + V_{ov}/(E_{sat}\cdot L_g))} \]
Ioff Off-Current
\[ V_{th,\text{eff}} = V_{th} - \frac{\text{DIBL} \cdot V_{dd}}{1000} \]
\[ I_{\text{off}} = 1.5\times10^{-7} \cdot \frac{W_{\text{eff}}}{L_g} \cdot 10^{-V_{th,\text{eff}}/(SS \times 10^{-3})} \quad \text{[A]} \]
Reference current 1.5×10⁻⁷ A calibrated from Sentaurus + 17-device benchmark (Loubet 2017 et al.)

The DIBL correction on Vth accounts for barrier lowering at high Vds = Vdd. The exponential 10^(-Vth/SS) is the standard subthreshold current formula from the textbook model (Taur & Ning). Clamped Vth,eff ≥ 0.05 V to prevent numerical blow-up.

For the full Id(Vgs, Vds) curve (subthreshold region):

\[ I_D = I_{ref} \cdot 10^{(V_{gs} - V_{th,eff})/SS} \cdot 10^{-3} \]
gm · gds Transconductance & Output Conductance
\[ g_m = \frac{2\,I_{on}}{V_{ov}} \quad \text{[S]} \]
\[ V_{\text{early}} = 15 \cdot \frac{L_g}{12}\,\text{[V]}, \qquad g_{ds} = \frac{I_{on}}{V_{\text{early}}} \quad \text{[S]} \]
Early voltage scales linearly with Lg (longer channel → better output resistance)

These are displayed in the parameter grid as mS/µm and µS/µm respectively, normalised to Weff.

🌡️ Temperature Correction Models

Based on Aruna Kumari 2025 and Martino 2025 nanosheet FET temperature measurements:

\[ V_{th}(T) = V_{th,25} - 0.0005\,(T - 25) \quad [\text{V}] \]
\[ SS(T) = SS_{25} + 0.19\,(T - 25) \quad [\text{mV/dec}] \]
\[ \mu_{eff}(T) \propto \left(\frac{300}{T+273}\right)^{1.5} \quad \text{(phonon scattering)} \]

The temperature sweep charts plot these three quantities continuously from 25°C to 125°C, with the current device state highlighted as a red dot.

🧠 SNN Performance Predictor — computeSNN(d, S)
LightGBM Surrogate — Linearised Coefficients

The full LightGBM model (500 trees, trained on 17,234 Sentaurus TCAD + SNN-simulator samples) is approximated by its first-order Taylor expansion around the calibration anchor points. This enables real-time evaluation in the browser without a Python backend.

\[ E\,[\text{fJ}] = 0.8 + 4.5\,I_{on}^{mA} + 0.001\,I_{off}^{pA} + 0.005\,SS + 0.003\,T \]
\[ L\,[\text{ms}] = 1.2 - 41.3\,I_{on}^{mA} + 0.014\,I_{off}^{pA} + 0.167\,SS + 0.01\,T \]
\[ A\,[\%] = 58.6 + 193.4\,I_{on}^{mA} - 0.077\,I_{off}^{pA} - 0.075\,SS - 0.05\,T \]
Ion in mA (total device, not per µm) · Ioff in pA · SS in mV/dec · T in °C

📈 Graph Generation — All 11 Charts
Chart 1 Transfer Characteristic Id–Vgs

X-axis: Vgs from 0 → Vdd (80 equally spaced points). Y-axis: Id (µA, log or linear). Three curves plotted simultaneously: TT, FF, SS corners.

For above-threshold (Vgs ≥ Vth):

\[ I_D = \frac{\beta\,(2V_{ov}-V_{ds})\,V_{ds}}{2\,(1 + V_{ov}/E_{sat}L_g)} \quad (V_{ds} < V_{dsat}) \]
\[ I_D = \frac{\beta\,V_{ov}^2}{1 + V_{ov}/E_{sat}L_g} \quad (V_{ds} \geq V_{dsat}) \]

For subthreshold (Vgs < Vth): DIBL-corrected exponential model. Toggle button switches Y-axis between log (Sentaurus-style) and linear scale.

Chart 2 Output Characteristic Id–Vds

X-axis: Vds from 0 → Vdd (60 points). Y-axis: Id (µA). Three corners at Vgs = Vdd. Uses the same Id_at(Vgs, Vds) function with the tanh-saturation approximation:

\[ I_D(V_{ds}) \approx I_{on} \cdot \tanh\!\left(\frac{V_{ds}}{V_{dsat}}\right) \]

This smoothly transitions from the linear region (slope ≈ gds) to saturation (Id ≈ Ion).

Canvas Live Band Structure Diagram

Drawn using Canvas 2D API. The energy bands are computed from Vth and Vdd:

  • Source Fermi level fixed at 0 eV reference
  • Channel barrier height = q·Vth (visualised as the peak in the conduction band)
  • Drain quasi-Fermi level drops by q·Vdd below source
  • Conduction band Ec drawn as a smooth cosine-interpolated curve through: source flat → barrier peak → gate region flat → drain drop
  • Valence band Ev drawn parallel to Ec with a 1.12 eV (Si bandgap) offset

Updates every time computeAll() is called via window.updateBandDiagram(Vth, Vdd).

Canvas SNN Spike Raster Plot

A real-time Leaky-Integrate-and-Fire (LIF) neuron simulation displayed as an animated raster:

\[ V_i[t+1] = V_i[t] \cdot \tau_{\text{decay}} + I_{\text{input}}[t] \]
\[ \text{if}\;V_i \geq V_{thresh}=1.0: \quad \text{spike, reset}\;V_i = 0 \]

Parameters: decay τ = 0.85, threshold = 1.0, Ninput ~ U(0, 0.45·tempRate). The energy E and latency L from computeSNN modulate:

  • Dot size: max(2.2, min(5.5, 2.0 + 0.6·E)) — higher energy → larger spikes
  • Glow radius: max(0, min(15, 2.0·E))
  • Latency label: displayed at bottom of canvas

10 neurons, 200 time-step history. Animates at 30 fps via setInterval. Each simulation step runs in < 0.1 ms.

Key caveats:

  • Energy/spike predictions assume ideal periphery — real systems may be 2–5× higher due to interconnect, SRAM, and routing capacitances.
  • The SNN surrogate was trained on LIF neuron architectures only. LSTM-SNN or adaptive-threshold variants are not covered.
  • Quantum transport effects (BTBT, direct source-to-drain tunnelling) are not modelled. These become significant for Lg < 8 nm.
  • The analytical model does not simulate layout parasitics (RC delays, fringing capacitances beyond the spacer model).
  • All results are pre-TCAD approximations and should be validated with full Sentaurus simulation before tape-out.
