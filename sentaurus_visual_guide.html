Overview & Functionality

Sentaurus Visual is the primary visualization tool for Synopsys TCAD. It provides an advanced graphical user interface and a powerful scripting engine to analyze 1D, 2D, and 3D simulation results.

  • Visualize: Device structures, I-V curves, scalar/vector fields (e.g., ElectrostaticPotential, CurrentDensity), depletion regions, and mesh properties.
  • Supported File Formats:
    • .tdr — TCAD Data Representation (structures and fields)
    • .plt — XY plot data (curves, electrical characteristics)
    • .plx, .csv — Tabular data export/import
    • .tif, .png — Image export

Getting Started

To start Sentaurus Visual from the command line:

svisual
svisual nsfet_des.tdr
svisual -x my_script.tcl

Alternatively, open files via the GUI using File > Load or by dragging and dropping .tdr / .plt files directly into the workspace.

Basic Operations

  • Linking Plots: Use the link_plots command or the GUI Link button to synchronize pan, zoom, and rotation across multiple plots. Useful when comparing two device states side-by-side.
  • Custom Buttons: You can assign frequently used Tcl scripts to custom toolbar buttons for rapid execution.
  • Navigation: Middle-click to pan, scroll to zoom, right-click and drag to rotate (3D). Use the Undo button (or undo command) to revert view changes.
  • Performance Options: For massive 3D meshes or dense XY plots, enable Fast Draw or adjust the Level of Detail (LoD) and subsampling in the Preferences to accelerate rendering.

Working with XY Plots

XY Plots are essential for visualizing electrical characteristics like I-V or C-V curves, or 1D extractions (cutlines) from devices.

Core Features

  • Cutline Plots: Extract 1D data along a line segment drawn across a 2D/3D field. The cutline can be dynamically dragged in the structure plot, updating the XY plot in real-time.
  • Curve Properties: Customize color, line style, markers, and thickness. Apply transformations (e.g., Absolute Value, Derivative, Integral) directly from the Data Explorer.
  • Axis & Legend Customization: Adjust scale (linear/logarithmic), range, padding, and tick precision. Support for Greek letters and symbols via XML-style tags (e.g., <i>m</i>).
  • Probing & Math: Use probe_curve to find intercepts or target Y values. Create new curves via math operations using create_curve -function.

Electrical Extraction Macro Example

# Extract Vth using constant current method
set vth [probe_curve -plot Plot_1 -curve Id -target_y 1e-7]
puts "Threshold Voltage: $vth V"

Working with 2D & 3D Plots

Advanced visualization of TCAD meshes and physical fields.

  • Visualizing Fields: Map scalar fields (e.g., DopingConcentration) to color palettes. Use Linear, Logarithmic, or Asinh scales. Vector fields (e.g., ElectricField) can be visualized using arrows.
  • Materials & Regions: Toggle visibility of specific materials or regions. Adjust translucency (alpha) to see inside 3D structures. Show/hide bulk elements, borders, and mesh nodes.
  • Junctions & Depletion: Sentaurus Visual automatically highlights PN junction lines and depletion region boundaries.
  • Cutlines & Cutplanes: Create Orthogonal, Free, Polyline, or Boundary cutlines. In 3D, use Cutplanes to slice through the device and expose internal field distributions.
  • Isosurfaces: Create 3D contours of constant scalar values using create_iso (e.g., mapping the exact boundary of an inversion channel).
  • Value Blanking: Hide mesh elements that meet specific criteria (e.g., ElectrostaticPotential < 0). Multiple constraints can be combined using Union or Intersection.
  • Streamlines: Visualize current flow paths using vector field integration and rakes.
  • Deformation: Warp the physical mesh based on mechanical displacement or stress tensors using set_deformation.

Automation & Scripting

Sentaurus Visual supports powerful automation via Tcl and Python scripts.

  • Running Scripts: Execute via File > Run Script or batch mode (svisual -b -x script.tcl).
  • Command History: Every GUI action generates a Tcl command. Save your session using File > Save Command File or save_plot_to_script.
  • Script Library: Automatically load custom macros by placing scripts with the lib_ prefix in your library path.

Example: Plotting an Id-Vg Curve

# Load the IV plot file
load_file nsfet_des.plt
create_plot -dataset nsfet_des -1d
create_curve -plot Plot_1 -dataset nsfet_des -axisX "gate OuterVoltage" -axisY "drain TotalCurrent"

# Format the plot
set_axis_prop -plot Plot_1 -axis y -type log
set_curve_prop -plot Plot_1 -curve "drain TotalCurrent" -color red -line_width 2
set_axis_prop -plot Plot_1 -axis x -title "Gate Voltage (V)"
set_axis_prop -plot Plot_1 -axis y -title "Drain Current (A)"
export_view -plot Plot_1 -format png -resolution 1920x1080 idvg_plot.png

Tcl Command Reference

A quick reference for the most commonly used Sentaurus Visual Tcl commands. Click a command name to copy it.

Command Description Example
load_file Loads a TDR, PLT, or other supported file into a dataset. load_file nsfet.tdr -name myData
create_plot Creates a new 1D, 2D, or 3D plot window. create_plot -dataset myData -2d
create_curve Plots XY data on a 1D plot or calculates math expressions. create_curve -plot Plot_1 -axisX time -axisY current
create_cutline Extracts a 1D line of data from a 2D/3D dataset. create_cutline -dataset myData -type x -position 0.5
create_cutplane Creates a 2D slice through a 3D dataset. create_cutplane -dataset my3D -type z -position 0.0
set_curve_prop Modifies curve attributes (color, style, markers). set_curve_prop -curve C1 -color blue -line_style solid
set_axis_prop Modifies axis scale, title, range, and ticks. set_axis_prop -axis y -type log -title "Id (A)"
set_legend_prop Adjusts legend visibility and positioning. set_legend_prop -plot Plot_1 -position {0.8 0.9}
set_field_prop Sets color maps, scale types, and visibility for fields. set_field_prop -field Doping -scale log -show_bands
set_material_prop Changes rendering properties for specific materials. set_material_prop -material Silicon -color gray
probe_curve Extracts specific X or Y values from a 1D curve. set vth [probe_curve -curve Id -target_y 1e-7]
probe_field Extracts a scalar/vector field value at specific (x,y,z). probe_field -dataset D1 -field Potential -coord {0 0 0}
integrate_field Integrates a field over a region, material, or boundary. integrate_field -dataset D1 -field eDensity -region Channel
create_iso Creates an isosurface/isoline at a specific field value. create_iso -dataset D1 -field Potential -value 0.5
export_view Saves the current plot as an image file (PNG, TIF). export_view -format png -resolution 1024x768 out.png
save_plot_to_script Exports the commands required to recreate a plot. save_plot_to_script -plot Plot_1 script.tcl
link_plots Synchronizes views (pan/zoom) between multiple plots. link_plots -plots {Plot_1 Plot_2}
overlay_plots Overlays curves or fields from different datasets. overlay_plots -plot1 P1 -plot2 P2
undo Reverts the last visualization operation. undo

Python Commands

Sentaurus Visual includes a Python scripting interface via the sv package. This allows you to leverage Python's extensive ecosystem (e.g., NumPy, Pandas, Matplotlib) alongside TCAD visualization.

  • Namespace: All commands are accessed via the sv module (e.g., sv.load_file(...)).
  • Help: Access built-in documentation using Python's help system: help(sv.create_plot).
  • Mapping: Python commands generally mirror Tcl commands identically, replacing spaces with function calls and hyphens with kwargs.

Python Example

import sv
# Load file and create plot
sv.load_file('device.tdr', name='myDevice')
sv.create_plot(dataset='myDevice', type='2d')

# Set field properties
sv.set_field_prop(field='ElectrostaticPotential', scale='linear')
sv.export_view(format='png', resolution='1920x1080', filename='potential.png')
